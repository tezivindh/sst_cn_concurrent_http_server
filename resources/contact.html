Server Information

Server Configuration

Default Host: 127.0.0.1 (localhost)

Default Port: 8080

Thread Pool Size: 10 threads

Connection Queue: 50 pending connections

Keep-Alive Timeout: 30 seconds

How to Start the Server

You can start the server with default settings or customize it with command-line arguments:

# Default configuration (localhost:8080, 10 threads)
python server.py
# Custom port
python server.py 8000
# Custom port and host
python server.py 8000 0.0.0.0
# Custom port, host, and thread pool size
python server.py 8000 0.0.0.0 20

Testing the Server

Testing GET Requests

Open your browser and navigate to:

  • http://localhost:8080/ - Home page
  • http://localhost:8080/about.html - About page
  • http://localhost:8080/logo.png - Download image
  • http://localhost:8080/sample.txt - Download text file

Testing POST Requests (JSON Upload)

Use curl or any HTTP client to send JSON data:

curl -X POST http://localhost:8080/upload \
  -H "Content-Type: application/json" \
  -H "Host: localhost:8080" \
  -d '{
    "name": "Test Data",
    "timestamp": "2024-03-15T10:30:00",
    "values": [1, 2, 3, 4, 5]
  }'

Expected response:

{
  "status": "success",
  "message": "File created successfully",
  "filepath": "/uploads/upload_20240315_103000_a7b9.json"
}

Testing Security Features

Path Traversal Protection

Try these requests - they should all return 403 Forbidden:

curl http://localhost:8080/../etc/passwd
curl http://localhost:8080/../../sensitive.txt
curl http://localhost:8080//etc/hosts

Host Header Validation

Request without Host header (should return 400):

curl -H "Host:" http://localhost:8080/

Request with wrong Host header (should return 403):

curl -H "Host: evil.com" http://localhost:8080/

Monitoring

The server provides comprehensive logging to the console. Watch the terminal output to see connection details, request processing, thread pool status, and any security violations.
